Assignment 9: Final Project Landing Page

Individual assignment.

Let's define a landing page as whatever fills the browser window after the URL is loaded. In the case of a single-page design, this will be whatever content fills the browser window before any interaction with the user. Animations are not necessary but all content should be there, as well as functional links to placeholder pages or, in the case of a single page design, placeholder sections.

This assignment requires that you implement a first iteration of your final project landing page.
